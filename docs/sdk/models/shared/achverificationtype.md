# AchVerificationType

Indicates the type of ACH verification being performed.


## Values

| Name            | Value           |
| --------------- | --------------- |
| `Basic`         | BASIC           |
| `AccountStatus` | ACCOUNT_STATUS  |
| `AccountOwner`  | ACCOUNT_OWNER   |
| `PreNoteCredit` | PRE_NOTE_CREDIT |
| `PreNoteDebit`  | PRE_NOTE_DEBIT  |

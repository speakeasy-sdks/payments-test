# ProcessorStatus

Codifies the state of the payment status from the payment processor. Examples include COMPLETED ,PAID, OVERPAID,ERRORED


## Values

| Name         | Value        |
| ------------ | ------------ |
| `Initiated`  | INITIATED    |
| `Completed`  | COMPLETED    |
| `Redirected` | REDIRECTED   |
| `Returned`   | RETURNED     |
| `Underpaid`  | UNDERPAID    |
| `Overpaid`   | OVERPAID     |
| `Paid`       | PAID         |
| `Voided`     | VOIDED       |
| `Error`      | ERROR        |
| `Expired`    | EXPIRED      |
